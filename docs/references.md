# References

This page is a curated list of public references that inspired the patterns in this repository.

## Tool use and agent prompting

- ReAct: reasoning + acting interleaving for tool use (Yao et al., 2022) — https://arxiv.org/abs/2210.03629
- Toolformer: self-supervised tool use learning (Schick et al., 2023) — https://arxiv.org/abs/2302.04761
- Tree of Thoughts: deliberate search over intermediate reasoning states (Yao et al., 2023) — https://arxiv.org/abs/2305.10601

## Safety and alignment

- Constitutional AI: harmlessness from AI feedback (Bai et al., 2022) — https://arxiv.org/abs/2212.08073

## Memory and long-horizon agents

- Generative Agents: memory + reflection + planning in simulations (Park et al., 2023) — https://arxiv.org/abs/2304.03442
- Voyager: lifelong learning agent with curriculum + skill library (Wang et al., 2023) — https://arxiv.org/abs/2305.16291

## Evaluation

- SWE-bench: real-world GitHub issue resolution benchmark — https://www.swebench.com/
- AgentBench: evaluating LLMs as agents across environments — https://arxiv.org/abs/2308.03688
- GAIA benchmark: general AI assistants with tool-use/web browsing — https://arxiv.org/abs/2311.12983
- HELM: holistic evaluation of language models — https://crfm.stanford.edu/helm/

## Risk management standards

- NIST AI RMF 1.0 — https://www.nist.gov/publications/artificial-intelligence-risk-management-framework-ai-rmf-10
